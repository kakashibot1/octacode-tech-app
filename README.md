# OCTACODE TECH APP 🚀

![Flutter](https://img.shields.io/badge/Flutter-2.10.0-blue?logo=flutter&logoColor=white)
![Dart](https://img.shields.io/badge/Dart-2.18.0-blue?logo=dart&logoColor=white)
![Licence](https://img.shields.io/badge/License-MIT-green)

**OCTACODE TECH APP** est une application mobile éducative conçue pour **apprendre la programmation et la technologie** de manière progressive, pratique et immersive.  
Développée par **Octavio Wina**, cette application est destinée à tous les passionnés de coding, débutants comme avancés.

---

## 💡 Objectifs du projet

- Fournir un **parcours structuré de 200 leçons** couvrant 4 technologies majeures
- Proposer une **interface sombre, moderne et ergonomique**
- Permettre aux utilisateurs de **pratiquer le code directement depuis leur smartphone**
- Offrir un **projet open-source** pour contribuer et partager des connaissances

---

## 📚 Contenu de l’application

L’application contient **200 leçons réparties en 4 lots** :

| Lot | Technologie | Nombre de leçons | Description |
|-----|------------|-----------------|------------|
| 1   | HTML + CSS | 50 | Bases du web, structure des pages, mise en page, typographie, responsive design |
| 2   | JavaScript | 50 | Variables, fonctions, DOM, événements, ES6, modules, Promises, API Fetch |
| 3   | Python     | 50 | Syntaxe, types, fonctions, classes, modules, décorateurs, gestion des erreurs |
| 4   | React.js   | 50 | Composants, props, state, hooks, routing, context, lazy loading, bonnes pratiques |

Chaque leçon comprend **un titre, une description, et des exemples de code clairs** pour une pratique directe.

---
